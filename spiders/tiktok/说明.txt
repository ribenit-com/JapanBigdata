海外抖音爬虫

spiders/tiktok/
├── config/                          # 配置文件目录
│   └── users.xml                    # 用户账号配置，存储邮箱和密码信息
│
├── example/                         # 示例代码和测试目录
│   ├── main.go                      # 主程序入口，演示爬虫使用方法
│   ├── test_data.go                # 数据查看工具，包含 Redis 和 MongoDB 数据查看方法
│   └── view_data_test.go           # 测试用例，包含数据查看和登录状态失效测试
│
├── model/                          # 数据模型定义目录
│   └── user.go                     # 用户信息结构体定义，包含：
│                                   # - 用户基本信息（邮箱、密码）
│                                   # - 浏览器信息
│                                   # - Cookie 数据
│                                   # - 登录状态和时间信息
│
├── scripts/                        # Python 脚本目录
│   └── get_ip.py                   # 获取当前 IP 的 Python 脚本
│
├── login.go                        # 登录相关功能实现
│                                   # - 表单填写
│                                   # - 验证码处理
│                                   # - 登录状态检查
│
├── tiktok_spider.go                # 爬虫核心实现
│                                   # - 浏览器控制
│                                   # - Cookie 管理
│                                   # - 数据存储（Redis/MongoDB）
│                                   # - 会话维护
│
└── utils.go                        # 工具函数
                                    # - Cookie 格式转换
                                    # - 数据处理辅助函数

主要功能模块：
1. 配置管理 (config/)
   - 用户账号管理
   - 登录凭证存储

2. 数据模型 (model/)
   - 用户信息结构
   - 浏览器信息定义
   - Cookie 数据格式

3. 核心功能 (*.go)
   - 自动登录
   - Cookie 管理
   - 会话维护
   - 数据持久化

4. 工具和脚本 (scripts/)
   - IP 检测
   - 环境检查

5. 示例和测试 (example/)
   - 使用示例
   - 数据查看
   - 功能测试
